# Руководство по деплою

## Подготовка к деплою

1. Убедитесь, что все зависимости установлены:
   ```bash
   npm install
   ```

2. Создайте файл `.env` с необходимыми переменными:
   ```env
   TELEGRAM_BOT_TOKEN=ваш_токен_бота
   TELEGRAM_CHAT_ID=ваш_id_чата
   PORT=3001
   ```

3. Соберите проект:
   ```bash
   npm run build
   ```

## Запуск сервера

Запустите сервер командой:
```bash
npm start
```

Или напрямую:
```bash
node server.js
```

Сервер будет доступен на порту, указанном в переменной `PORT` (по умолчанию 3001).

## Настройка Telegram бота

1. Найдите в Telegram бота @BotFather
2. Отправьте ему команду `/newbot`
3. Следуйте инструкциям для создания нового бота
4. После создания BotFather предоставит вам токен для бота
5. Найдите в Telegram бота @userinfobot и отправьте ему любое сообщение, чтобы получить ваш ID чата
6. Добавьте токен и ID чата в файл `.env`

## Проверка работоспособности

После настройки Telegram бота вы можете протестировать его работу, отправив POST-запрос на `/api/telegram-webhook`:

```bash
curl -X POST http://ваш_домен/api/telegram-webhook \
  -H "Content-Type: application/json" \
  -d '{"name":"Тест","email":"test@example.com","phone":"+79991234567","message":"Тестовое сообщение"}'
```

Вы должны получить ответ:
```json
{"success":true,"message":"Сообщение успешно отправлено!"}
```

И в вашем Telegram-чате должно появиться сообщение от бота.
